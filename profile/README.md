# WPI Combat Robotics
Student-run club at Worcester Polytechnic Institute that designs and builds combat robots for competition. We train members in CAD, 3D printing, manufacturing, electronics, and testing, and compete in 1lb, 3lb, 12lb, and 30lb classes nationwide. We also host on-campus antweight and beetleweight events for WPI and local builders.
